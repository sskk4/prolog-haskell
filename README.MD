# Haskell i Prolog – Programowanie Deklaratywne

## Haskell – Programowanie Funkcyjne

### Pliki podstawowe (1-4.hs)

| Plik  | Temat | Zakres |
|-------|--------|--------|
| **1.hs** | Podstawy | Funkcje, arytmetyka (`f x = 2*x + 1`) |
| **2.hs** | Operatory i funkcje matematyczne | Sekcje, `sphereVolume`, `coneVolume` |
| **3.hs** | Guards i rekurencja | `factorial`, operatory własne (`><`) |
| **4.hs** | Listy i przetwarzanie | `map`, `filter`, list comprehensions, liczby pierwsze |

| Plik | Temat | Zakres |
